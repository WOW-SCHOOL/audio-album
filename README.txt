Как пользоваться мини‑сайтом-плеером
=====================================

1) Скопируйте ваши MP3/OGG файлы в папку `tracks/`.
2) Откройте файл `playlist.json` и пропишите список треков:
   - `title` — название
   - `artist` — автор/альбом (необязательно)
   - `file` — относительный путь к файлу в `tracks/`

3) Проверьте локально:
   - запустите простой сервер: `python -m http.server 8000` и откройте http://localhost:8000

4) Выложите бесплатно как статический сайт:
   - **GitHub Pages**: создайте репозиторий, залейте файлы, включите Pages (ветка `main`, папка `/`).
   - **Netlify / Vercel**: перетащите папку в веб‑интерфейс — получите URL.

5) Вставьте в Miro через iFrame:
   <iframe src="https://ВАШ-ДОМЕН/index.html" width="100%" height="520" frameborder="0" allow="autoplay"></iframe>

Примечания:
- Автоплей может блокироваться браузером — это нормально.
- Сайт работает без бекенда, только статические файлы.
